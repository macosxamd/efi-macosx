Select the ROM file according to the virtual machine's bit version: EFI32 for 32-bit and EFI64 for 64-bit. Place the ROM file in the virtual machine's folder. Then, use Notepad to add the following text to the end of the virtual machine's vmx file:

efi32.filename = "EFI32-MACOS.ROM" (32-bit)
efi64.filename = "EFI64-MACOS.ROM" (64-bit)
